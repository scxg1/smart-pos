# Smart POS System — فحص شامل للميزات

## معلومات عامة

| البند | التفاصيل |
|---|---|
| **اسم النظام** | نقطة البيع الذكية — Smart POS |
| **الإصدار** | 1.0.0 |
| **التقنيات** | React 18 + TypeScript + Zustand + Tailwind CSS + Vite |
| **الخادم** | Express.js + sql.js (SQLite) |
| **سطح المكتب** | Electron 28 |
| **الذكاء الاصطناعي** | GLM-5.1 عبر Z.AI API |
| **اللغة** | العربية (RTL) بالكامل |
| **قاعدة البيانات** | SQLite (في الذاكرة مع حفظ تلقائي على القرص) |

## قاعدة البيانات — الجداول

| الجدول | الوصف | الأعمدة الرئيسية |
|---|---|---|
| `products` | كتالوج المنتجات والمخزون | id, name, category, barcode, cost_price, selling_price, stock, unit, image_path, created_at |
| `customers` | سجلات العملاء | id, name, phone, total_purchases |
| `sales` | فواتير البيع | id, customer_id, total, cost_total, profit, discount, tax, payment_method, note, created_at |
| `sale_items` | عناصر الفاتورة | id, sale_id, product_id, name, quantity, unit_price, cost_price, subtotal |
| `settings` | إعدادات النظام (key-value) | key (PK), value |
| `chat_sessions` | جلسات محادثة AI | id, title, created_at, updated_at |
| `chat_messages` | رسائل محادثة AI | id, session_id (FK), role, content, created_at |
